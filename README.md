# BlueIsland 留言板

一个治愈风格的岛屿主题留言板系统，包含 AI 审核、AI 回响、情绪感知等功能。

## 技术栈

- **前端**: React 19, Vite, TailwindCSS
- **后端**: .NET 9, FastAPI (ASP.NET Core), SqlSugar ORM
- **数据库**: MySQL

## 功能特性

- AI 岛屿天气与情绪感知
- 海浪白噪音 ASMR 叠播
- 视觉潮汐系统（昼夜滤镜）
- AI 内容审核
- AI 回响生成
- 留言发布与查询
- 私密留言加密

## 快速开始

### 前端

```bash
cd my-react-app
npm install
npm run dev
```

### 后端

```bash
cd web/server
dotnet run --project BlueIsland.Api
```
